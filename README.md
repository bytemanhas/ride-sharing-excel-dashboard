# ride-sharing-excel-dashboard
Interactive Excel dashboard analyzing ride-sharing demand and revenue. The project uses pivot tables, pivot charts, KPIs, and slicers to visualize ride demand, revenue distribution, trip distance patterns, and driver performance across multiple cities.


# 🚗 Ride Sharing Data Analysis (Excel)

## 📌 Project Overview
This project analyzes ride-sharing data (similar to Uber/Ola) using Microsoft Excel to understand demand patterns, pricing trends, and operational performance. The goal is to extract insights that help improve efficiency and decision-making.

---

## 🎯 Objective
The objective of this project is to:
- Analyze ride demand across cities and time periods
- Identify peak hours and high-demand locations
- Evaluate revenue and pricing trends
- Improve operational efficiency

---

## 🛠️ Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Data Cleaning & Formatting
- Charts & Visualization

---

## 📂 Dataset Description
The dataset contains ride-sharing data with the following fields:
- Ride ID
- Customer ID
- Driver ID
- Pickup Location
- Drop Location
- Ride Distance
- Fare Amount
- Ride Time
- City
- Ride Date

---

## 📊 Analysis Performed

### 1. Total Rides & Revenue
- Calculated total rides using COUNT
- Calculated total revenue using SUM

### 2. Rides by City
- Used Pivot Tables to analyze ride distribution across cities

### 3. Peak Hour Analysis
- Extracted hour from ride time
- Identified peak demand periods using Pivot Tables

### 4. Average Fare Analysis
- Calculated average fare per ride

### 5. Location-Based Demand
- Identified high-demand pickup locations

---

## 🔍 Key Insights
- Peak demand occurs during morning and evening hours
- Certain locations consistently have higher ride demand
- Revenue is higher in urban areas
- Ride patterns vary based on time and location

---

## 💡 Business Recommendations
- Increase driver availability during peak hours
- Optimize pricing during high-demand periods
- Focus on high-demand locations for better efficiency
- Improve operations in low-performing areas

---

## 📌 Conclusion
This project demonstrates how Excel can be used for data analysis, enabling businesses to understand ride demand patterns and improve operational decisions.
